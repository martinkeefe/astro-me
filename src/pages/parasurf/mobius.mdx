## Möbius Strip

The Möbius strip or band is a topological object and so it does not have a
defined geometric shape. However, this particular version is rather pleasing.

<div id="ps-shape"></div>

### Parameterization

> $$\begin{align} x &= r\,(1 + w\,v \cos\tfrac{u}{2}) \cos u&	u &\in [-\pi,\pi)\\ y &= r\,(1 + w\,v \cos\tfrac{u}{2}) \sin u&	v &\in [-\tfrac{1}{2},\tfrac{1}{2}]\\ z &= w\,v \sin\tfrac{u}{2} \end{align}$$

This parameterization creates a Möbius strip of width $w$ that springs from
a <span class="extra-0">centre circle</span> of radius $r$ that lies in the
XY plane and is centered at (0, 0, 0).
The parameter $u$ runs around the strip while $v$ moves from one edge to
the other. It is easy to see that there is a single <span class="extra-1">boundary</span>.
